# 自动化审稿系统 — 笔记

## 项目概述

中国政法大学研究生学报私法组投稿自动处理工具，位于 `自动化审稿系统/后端审稿系统/journal_automation/`。

### 架构

- **mail.py** — 邮件获取（IMAP）与解析
- **metadata.py** — 稿件元数据提取（标题、学科、作者等）
- **workflow.py** — 主流程编排
- **storage.py** — 附件归档与目录管理
- **templates.py** — 审稿评分表生成与邮件草稿
- **workbook.py** — Excel台账操作
- **state.py** — SQLite状态管理
- **config.py** — 配置加载
- **models.py** — 数据模型
- **utils.py** — 工具函数

## 2026-05-06 设计决策

### 三模块扩展方案

基于用户5步工作流设计的三个新模块：

1. **sync-assignments** — 派稿分派记录
   - 扫描"审稿回复"文件夹（IMAP）/或读取本地 `派稿及回复/` 目录
   - 记录：派稿时间、作者姓名、一审编辑姓名、学科
   - 编辑身份在派稿时确定（而非从评分表猜测）
   - 需要用户确认分派顺序

2. **sync-reviews** — 编辑审稿回复导入
   - 扫描审稿回复中编辑回复的附件
   - 解析审稿评分表.docx（总分、审定意见、论文评价）
   - 按时间+作者名匹配到派稿记录，确定编辑身份
   - 更新 Excel 审稿状态

3. **sync-notifications** — 作者通知同步
   - 扫描 1/2/3 审程序文件夹（已发送通知）
   - 按时间+作者名匹配投稿记录
   - 更新 Excel 通知状态

### 编辑联系人配置

已收集20位编辑的邮箱，覆盖7个学科方向，写入 `automation.config.json` 的 `editor_contacts` 字段。

### 本地派稿目录结构

`2. 派稿及回复/YYYYMMDD/` 目录结构：

```
20260401/
├── 【一审退稿】民法-王蕙巧-论文标题/
│   ├── 审稿评分表.docx
│   ├── 【批注版】xxx.docx
│   └── 原稿.doc
├── 【一审返修】商法-陈海鑫-论文标题/
│   └── ...
```

文件夹命名规则：`【审稿结论】学科-编辑姓名-论文标题`
